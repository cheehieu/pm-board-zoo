## Overview
This repository includes (for each EVM):
- Altium library, schematic, and board files
- PCB gerbers and drill files
- Bill of materials
- AfternoonCape config file

Board Zoo Animals:
- Duck (AM335x GP EVM)
- Dog (AM437x GP EVM Alpha)
- Cat (AM43xx X EVM Alpha)
- Turtle (AM43xx X EVM Beta)
